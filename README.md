# POKAD - Persian Offline Knowledge AI & Digital Assistant

POKAD is an offline Persian AI assistant based on Llama-2-7B. It supports bidirectional translation, web search, short-term memory, and a PyQt GUI.

## Features
- Fully offline (except optional web search)
- Persian conversation
- Automatic English-Persian translation
- SQLite history storage
- Remembers user name
- PyQt6 graphical interface
